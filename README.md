# 🌿 Project Pokos Travi

Одностраничный сайт для локального сервиса по покосу травы и бытовым работам.  
Цель проекта — предоставить клиентам удобный способ узнать услуги и быстро связаться с исполнителями.

---

## 📌 Проблема

- Отсутствует онлайн-присутствие сервиса  
- Клиенты не видят список услуг сразу  
- Нет примеров выполненных работ  
- Коммуникация происходит через звонки и мессенджеры вручную  

Это снижает доверие и усложняет привлечение новых клиентов.

---

## 💡 Решение

- Одностраничный сайт с блоками услуг  
- Галерея выполненных работ  
- Быстрые кнопки для связи через WhatsApp и Telegram  
- Адаптивная верстка для мобильных и десктоп устройств  

Сайт служит как точка входа для клиентов и упрощает коммуникацию.

---

## ⚙️ Функционал

- Два основных блока услуг: покос травы и разнорабочие  
- Галерея работ с фотопримерами  
- Кнопки быстрой связи: WhatsApp и Telegram  
- Адаптивная верстка (mobile / desktop)  

---

## 🛠 Технологии

HTML5 • CSS3 • JavaScript • Vite
